列表字典前面加星号

一个星号将列表拆分成独立的参数作为形参传入函数；两个星号将字典拆分成独立的形参传入函数

def add(a, b):
    return a + b

data = [1, 2]
add(*data) # 等同于 add(1, 2)

data = {'a':3, 'b':4}
add(**data) # 等同于 add(a=3,b=4)

end

yield

用于生成迭代器，可以调用 next() 函数不断得到下一个值。每次调用都会在 yield 的地方返回，下次调用的时候，会接着上次返回地 方的下一条语句继续执行，而不是从头开始执行。

def xrange(n):
    print("func xrange")
    i = 0
    while i<n:
        yield i
        i += 1

for i in xrange(5):
    print(i)

'''
func xrange
0
1
2
3
4
'''

g = xrange(5)
next(g) # 0
next(g) # 1
next(g) # 2
next(g) # 3

np.newaxis

在 np.newaxis 所在的位置增加一个维度

>>> import numpy as np
>>> a = np.array([1,2,3])
>>> a.shape
(3,)
>>> b = a[:, np.newaxis]
>>> b.shape
(3, 1, 1)
>>> b
array([[1],
       [2],
       [3]])
# 这里大致可以看到 ... 的作用，表示剩余的所有
>>> c = a[np.newaxis, ..., np.newaxis, np.newaxis]
>>> c.shape
(1, 3, 1, 1)
>>> c
array([[[[1]],

        [[2]],

        [[3]]]])

end

np.repeat

可以用于图像上采样

repeat(a, repeats, axis=None)
'''
按照指定 axis 重复 repeats 次生成新的数组；
不知道 axis 时，生成数组为 flat ，即 1 维；
可以让不同的位置 repeats 不同的次数
'''

Examples
--------
>>> x = np.array([[1,2],[3,4]])
>>> np.repeat(x, 2)
array([1, 1, 2, 2, 3, 3, 4, 4])
>>> np.repeat(x, 3, axis=1)
array([[1, 1, 1, 2, 2, 2],
       [3, 3, 3, 4, 4, 4]])
>>> np.repeat(x, [1, 2], axis=0)
array([[1, 2],
       [3, 4],
       [3, 4]])
